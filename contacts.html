Tienes preguntas?

¿Necesitas una cotización de nuestros servicios o requieres de un empleo ? utilice el siguiente formulario para enviar un ticket a nuestro equipo de soporte. Asegúrese de seleccionar el motivo de contacto adecuado para que podamos filtrar su mensaje correctamente.

Haremos todo lo posible para comunicarnos con usted lo antes posible. Por favor, espere hasta dos días hábiles para una respuesta, debido al volumen de consultas recibidas.

Si le faltan correos electrónicos de nuestra parte, verifique la carpeta de correo no deseado de su cuenta de correo electrónico para asegurarse de que el mensaje no se haya filtrado. Si el mensaje se filtró, puede encontrar una opción para «Marcar como bueno«, «No spam«, «No basura» o «Agregar remitente a la bandeja de entrada«. Esto ayudará a recibir futuros correos electrónicos de Integra Soluciones Globales
